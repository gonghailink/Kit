## 🎯 新增資料夾功能

### **1. 新增根資料夾（主要資料夾）**

#### **方式一：當 Tab 還沒有資料夾時**
```
┌─────────────────────────────────────┐
│  此 Tab 尚無資料夾                   │
│                                     │
│     [➕ 建立資料夾]                  │
└─────────────────────────────────────┘
```
點選中央的「建立資料夾」按鈕

#### **方式二：在現有資料夾列表底部**
```
┌─────────────────────────────────────┐
│  📁 資料夾 1                         │
│    🔖 書籤 1                         │
│    🔖 書籤 2                         │
├─────────────────────────────────────┤
│  📁 資料夾 2                         │
│    🔖 書籤 3                         │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│      [➕ 新增資料夾]                 │ ← 新增的按鈕
└─────────────────────────────────────┘
```
滾動到資料夾列表底部，點選虛線框的「新增資料夾」按鈕

### **2. 新增子資料夾（嵌套資料夾）**

#### **從資料夾下拉選單新增**
```
┌─────────────────────────────────────┐
│  📁 資料夾 1              [➕] [⋮]   │
│                                     │
│  點選 [⋮] 顯示選單：                 │
│  ┌──────────────────────┐          │
│  │ ➕ 新增子資料夾       │ ← 新增   │
│  │ ✏️  編輯              │          │
│  │ 🗑️  刪除              │          │
│  └──────────────────────┘          │
└─────────────────────────────────────┘
```

#### **子資料夾也可以繼續新增子資料夾**
```
📁 主資料夾
  └─ 📁 子資料夾 1
      └─ 📁 子資料夾 2
          └─ 📁 子資料夾 3
              └─ 🔖 書籤
```
支援無限層級的嵌套結構

## 🔄 完整的資料夾操作流程

### **場景 1：從零開始建立結構**
1. 建立 Tab（例如：「工作」）
2. 在 Tab 下建立根資料夾（例如：「開發工具」）
3. 在「開發工具」下新增子資料夾（例如：「前端框架」）
4. 在「前端框架」內新增書籤（例如：「React 官網」）

### **場景 2：組織現有書籤**
1. 已有多個資料夾和書籤
2. 點選資料夾下拉選單 → 新增子資料夾
3. 拖放書籤到新的子資料夾
4. 調整資料夾順序

### **場景 3：重新整理結構**
1. 點選資料夾下拉選單 → 編輯（重新命名）
2. 點選資料夾下拉選單 → 刪除（移除不需要的資料夾）
3. 拖放排序重新組織

## 💾 資料庫結構

### **Folders 表格**
```sql
create table public.folders (
  id uuid primary key,
  user_id uuid references auth.users not null,
  tab_id uuid references public.tabs on delete cascade not null,
  parent_id uuid references public.folders on delete cascade,  -- 支援嵌套
  title text not null,
  is_collapsed boolean default false,
  sort_order float default 0,
  created_at timestamp with time zone not null
);
```

**關鍵欄位：**
- `parent_id`: 用於建立階層關係
  - `null` = 根資料夾
  - 有值 = 子資料夾
- `on delete cascade`: 刪除父資料夾時自動刪除所有子項

## 🔧 API 端點

### **檔案位置**
[app/routes/api.folders.tsx](app/app/routes/api.folders.tsx)

### **支援的操作**

#### **1. 建立資料夾**
```typescript
POST /api/folders
{
  intent: "create",
  tab_id: "uuid",
  parent_id: "uuid" | null,  // null = 根資料夾
  title: "資料夾名稱"
}
```

#### **2. 更新資料夾**
```typescript
POST /api/folders
{
  intent: "update",
  id: "uuid",
  title: "新名稱",
  is_collapsed: "true" | "false"
}
```

#### **3. 刪除資料夾**
```typescript
POST /api/folders
{
  intent: "delete",
  id: "uuid"
}
```

#### **4. 重新排序**
```typescript
POST /api/folders
{
  intent: "reorder",
  ids: ["uuid1", "uuid2", "uuid3"],
  sortOrders: [1000, 2000, 3000]
}
```

## 🚀 快速開始

### **1. 建立第一個資料夾**
1. 登入後建立或選擇一個 Tab
2. 點選「建立資料夾」按鈕
3. 輸入資料夾名稱（例如：「前端開發」）
4. 點選「建立」

### **2. 建立子資料夾**
1. 找到要新增子資料夾的資料夾
2. 點選資料夾標題旁的 [⋮] 按鈕
3. 選擇「新增子資料夾」
4. 輸入名稱並建立

### **3. 在資料夾內新增書籤**
1. 點選資料夾標題旁的 [➕] 按鈕
2. 或從下拉選單選擇「新增書籤」
3. 填寫書籤資訊並儲存
